pyPDAF.PDAFomi.init_local¶

pyPDAF.PDAFomi.init_local() → None¶

Allocating an array of obs_l derived types instances.

This function initialises the number of observation types for each local analysis domain, which should be called at the start of the local analysis loop in py__init_dim_obs_l_pdaf().
