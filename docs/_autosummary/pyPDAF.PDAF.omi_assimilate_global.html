pyPDAF.PDAF.omi_assimilate_global¶

pyPDAF.PDAF.omi_assimilate_global()¶

Global filters except for 3DVar for a single DA step using diagnoal observation error covariance matrix.

See pyPDAF.PDAF.omi_assimilate_enkf_nondiagR(), or pyPDAF.PDAF.omi_assimilate_global_nondiagR(), or pyPDAF.PDAF.omi_assimilate_nonlin_nondiagR() for non-diagonal observation error covariance matrix.

Here, this function call is used for global stochastic EnKF [1], E(S)TKF [2], SEEK [2], SEIK [2], NETF [3], and particle filter [4]. The filter type is set in pyPDAF.PDAF.init(). This function should be called at each model time step.

The function is a combination of pyPDAF.PDAF.omi_put_state_global() and pyPDAF.PDAF.get_state().

This function executes the user-supplied functions in the following sequence:

  1. py__collect_state_pdaf

  2. py__prepoststep_state_pdaf

  3. py__init_dim_obs_pdaf

  4. py__obs_op_pdaf (for ensemble mean)

  5. py__obs_op_pdaf (for each ensemble member)

  6. core DA algorithm

  7. py__prepoststep_state_pdaf

  8. py__distribute_state_pdaf

  9. py__next_observation_pdaf
