pyPDAF.deallocate¶

pyPDAF.deallocate() → None¶

Finalise the PDAF systems including freeing some of the memory used by PDAF.

This function cannot be used to free all allocated PDAF memory. Therefore, one should not use pyPDAF.PDAF.init() afterwards.
