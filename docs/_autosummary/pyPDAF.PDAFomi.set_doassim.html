pyPDAF.PDAFomi.set_doassim¶

pyPDAF.PDAFomi.set_doassim(i_obs: int, doassim: int) → None¶

Setting the doassim attribute of obs_f for i-th observation type. This property must be explicitly set for OMI functionality.

Properties of obs_f are typically set in user-supplied function py__init_dim_obs_pdaf.

This is by default set to 0, which means that the given type of observation is not assimilated in the DA system.

Parameters:

  • i_obs (int) – index of observation types

  • doassim (int) –

    1. do not assimilate;

    2. assimilate the observation type
