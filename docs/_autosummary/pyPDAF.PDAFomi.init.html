pyPDAF.PDAFomi.init¶

pyPDAF.PDAFomi.init(n_obs: int) → None¶

Allocating an array of obs_f derived types instances.

This function initialises the number of observation types, which should be called at the start of the DA system after pyPDAF.PDAF.init().

Parameters:

n_obs (int) – number of observations
