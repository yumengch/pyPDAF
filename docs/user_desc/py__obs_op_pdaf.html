py__obs_op_pdaf¶

py__obs_op_pdaf(step: int, dim_p: int, dim_obs_p: int, state_p: np.ndarray, m_state_p: np.ndarray) → np.ndarray¶

Apply observation operator

This function computes \(\mathbf{H} \mathbf{x}\), where \(\mathbf{H}\) is the observation operator and \(\mathbf{x}\) is state vector.

Parameters¶

stepint

Current step

dim_pint

Dimension of state vector

dim_obs_p: int

Dimension of observation vector

state_p: np.ndarray[np.float, dim=1]

State vector. shape: (dim_p,)

m_state_p: np.ndarray[np.float, dim=1]

Observed state vector. shape: (dim_obs_p,)

Returns¶

m_state_p: np.ndarray[np.float64, dim=1]

Observed state vector. shape: (dim_obs_p,)
